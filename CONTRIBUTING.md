# Contributing

Pointr is a personal portfolio project by Chris Butler. Pull requests are not accepted.

Bug reports and feature requests are welcome via [GitHub Issues](https://github.com/notchrisbutler/pointr/issues).

## Filing a Bug Report

A good bug report helps reproduce and diagnose the problem quickly. Please include:

- **Browser and version** — e.g., Chrome 120, Safari 17.2
- **Device** — desktop or mobile, and screen size if relevant
- **Steps to reproduce** — a minimal, numbered sequence of actions that triggers the bug
- **Expected behavior** — what you expected to happen
- **Actual behavior** — what actually happened
- **Number of players** — how many people were in the session when the issue occurred

## Filing a Feature Request

When requesting a feature, please describe:

- **The use case** — what you are trying to do and in what context
- **Why it would be useful** — how it improves the planning poker experience for agile teams
