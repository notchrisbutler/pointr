# Security Policy

## Supported Versions

Only the latest deployed version of Pointr receives security fixes. Older versions are not supported.

| Version | Supported |
|---------|-----------|
| Latest  | Yes       |
| Older   | No        |

## Reporting a Vulnerability

To report a security vulnerability, send an email to **me@chrisbutler.dev** with a description of the issue, steps to reproduce, and any relevant details about the impact.

Please do not disclose the vulnerability publicly until a fix has been released.

**Response timeline:**

- **48 hours** — acknowledgment of your report
- **7 days** — fix target for confirmed vulnerabilities

After a fix is released, you are welcome to disclose the details publicly.
